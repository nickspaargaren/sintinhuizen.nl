# Sintinhuizen.nl

🇳🇱 De website voor de jaarlijkse intocht van Sinterklaas in de gemeente Huizen. In opdracht van St. Intocht Sint Nicolaas Huizen.

## How to contribute

This website is open source, which means that anyone can suggest changes and contribute improvements. You need a [GitHub account](https://github.com/signup) to get started.

### For everyone

No technical setup needed - for updating information on the website.
